== Description ==
Tema vuoto con jquery, bootstrap 3.3.4 e font-awesome-4.3.0.
Usa questo tema per creare fantastici siti con i tuoi custom css e javascript ;)

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in "Tema vuoto" in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
5. Navigate to Appearance > Customize in your admin panel and customize to taste.
